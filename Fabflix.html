Raymond Sy

Fabflix

Created an online Movie Store hosted on a Tomcat Web Server running on an AWS Linux instance. With MySql, populated the Database with User/Credit Credentials and Movie Data. Implemented full-text search for movies and stored-procedures for faster data retrieval. Utilized JDBC to communicate between Server and Database. Created a user-friendly website with HTML/CSS/Bootstrap framework and used AJAX/JSP to send/request data from Server. Applied HTTPS and Google reCaptcha to increase security. Scaled the entire project by implementing Connection pooling, Master-Slave, and Load-Balancing. In addition, created an Android Mobile version of Fabflix, which featured the login and search capabilities of the website. .
